This blue header area has a set width of 960px, is floated left, requiring a "clearfix" on the container.

If you resize your browser window so that it's smaller than 960px, you should see a horizontal scrollbar appear on the window.

Use the link below to toggle overflow: hidden on the container element:

Click to Add "overflow: hidden"

After you add overflow: hidden, you'll notice the container (which has an exaggerated 5px border) expands to properly contain the floated child element. But then the horizontal scrollbar on the browser window disappears, and some of the header content may be cut off and inaccessible. Your window must be smaller than 960px in order to see this problem.
